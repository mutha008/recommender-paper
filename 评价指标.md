## 评价指标

- PV

- UV

- Visit 

  - 整个浏览行为

- LandingPage

  - 进入的第一个落地页

- ExitPage

  - 退出前最后一个页面

- BounceRate 跳出率

  - 衡量黏性   =跳出数/浏览量

- Impressions 展现

- Hit 服务器打点数

- Conversion Rate 转化率

- Duration 停留时间

- New Visitor 初次访问者

  - 一段时间内第一次访问，即为新的初防者

- Return Visitor 回访者

  - 类似留存

- Referrer 访问来源

  - 流量来源

- New users 新用户

- Active Users 活跃用户

- Updated Users 升级用户数

- Retention Rate 留存率

- Total Users 总用户

- K-Factor K因子  

  - 每个用户发出邀请/转化的新用户
